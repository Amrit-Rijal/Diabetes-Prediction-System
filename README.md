# Diabetic Prediction System

A machine learning project that predicts whether a person has diabetes based on medical input data.

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Dataset

- Pima Indians Diabetes Dataset

## Machine Learning Steps

- Import libraries
- Load dataset
- Data preprocessing
- Split data into training and testing sets
- Standardize features
- Train Support Vector Machine (SVM) model
- Evaluate model accuracy
- Save the trained model
- Predict diabetes using new input data

## Project Files

```
Diabetic_Prediction_System.ipynb
diabetes.csv
trained_model.sav
README.md
```

## How to Run

1. Clone the repository.
2. Install the required libraries.
3. Open `Diabetic_Prediction_System.ipynb`.
4. Run all the cells.
5. Enter input values to get the prediction.

## Output

The model predicts whether the person is:

- Diabetic
- Non-Diabetic
